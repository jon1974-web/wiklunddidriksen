## Getting Started

Simply open `index.html` in your web browser to view the homepage.

### Local Development

You can use any local server to run the project:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (if installed)
npx serve

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## Customization

### Colors

Edit the CSS variables in `styles.css` to customize the color scheme:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... */
}
```

### Content

- Edit `index.html` to update text content
- Modify sections in the HTML file
- Update contact links with your actual social media/email

### Styling

- All styles are in `styles.css`
- Responsive breakpoints are defined in the media queries section
